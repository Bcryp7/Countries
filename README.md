### Countries of the World - JSON Format

#### A list of all countries with their information.

##### info:

- Country Code
- Native Name
- Phone Code
- Continent
- Capital
- Currency
- Languages

##### Format:

```
"data": {

    "Andorra": {
        "code": "AD",
        "native": "Andorra",
        "phone": "376",
        "continent": "EU",
        "capital": "Andorra la Vella",
        "currency": "EUR",
        "languages": [
          "ca"
        ]
      }
}
```
